4- Quais as vantagens e desvantagens os padrões da Web definidos pela W3C?

8- Sobre CSS3 (responda com exemplos):
		A- O que é?
			É uma linguagem que define o estilo da informação apresentada em
			uma página Web.
			
		B- Como escrever um estilo para um ID?
			Utilizando a seguinte forma. #exemplo {estilos... } ---> Ex: #menu {margin: 0 auto;}
			
		C- Como escrever um estilo para uma classe?
			Utilizando a seguinte forma. .exemplo {estilos... } ---> Ex: .menu {margin: 0 auto;}
			
		D- Como escrever um estilo para uma tag dentro de uma classe?
			Utilizando a seguinte forma. .exemplo "tag desejada" {estilos... } ---> Ex: .exemplo h1 {color: red;
			}
		E- Como escrever um estilo para uma tag?
			utilizando a seguinte forma. "tag {estilo... } ---> Ex: body {background-color: Blue;}
			
		F- O que é Box Model?
			E um elemento que serve para descrever o conteudo de um certo elemento sendo esse conteudo o espaço, o tamanho da borda etc.
			
		G- O que é Floating?
			e uma propriedade float do CSS que determina que um elemento deve ser retirado do seu fluxo normal e colocado ao longo do lado direito ou esquerdo
			do seu containêr, onde textos e elementos em linha irão se posicionar ao seu redor.
			
		H- O que o Floating muda do desenvolvimento Web atual em relação ao modo antigo (com uso de tabelas na construção do site)?
			Criação de sites mais dinamicos e melhor trabalhados.
